Personal Project

AlphaOne

Fine-tuned BERT for per-stock sentiment classification from Reddit, with LoRA adaptation and a real-time React dashboard.

Full-Stack Fine-Tuned BERT Aspect-Based Sentiment LoRA
View on GitHub Back to Portfolio

The Vision

Retail investors are drowning in social media noise. AlphaOne cuts through it with a custom fine-tuned BERT model that classifies sentiment per stock within the same sentence. When a post says "AAPL is great but TSLA is doomed," the model correctly labels AAPL as bullish and TSLA as bearish, something off-the-shelf sentiment models cannot do.

The system ingests posts from financial subreddits through a reproducible data pipeline into PostgreSQL, processes each post through a multi-stage NLP pipeline with spaCy tokenization, entity replacement, subject-conditioned inference, and topic tagging, then surfaces ticker sentiment trends through a React dashboard.

Built with production-grade data engineering principles. Concurrent-safe, idempotent batch ingestion with content versioning ensures no duplicate or corrupted records, even under parallel worker loads.

Technical Highlights

  • 4-Service Architecture

    React, Spring Boot, Celery/Redis, and PostgreSQL, deployed via Docker Compose.

  • Fine-Tuned BERT for Aspect-Based Sentiment

    LoRA-adapted model classifying per-stock sentiment via entity replacement. 47% accuracy improvement over the pretrained baseline.

  • Idempotent Data Pipeline

    Concurrent-safe batch ingestion with content versioning and deduplication. Fully reproducible from scratch.

  • Interactive Dashboard

    Real-time sentiment charts, ticker drill-downs, daily trends, and topic-based filtering.

Under The Hood

Backend API

Spring Boot serves the REST API with paginated endpoints for posts, sentiment scores, and aggregated trends. The API layer handles request validation, caching, and CORS configuration for the React frontend.

Worker Pipeline

Celery workers with Redis broker handle the heavy lifting: Reddit scraping with rate-limit backoff, NLP enrichment (spaCy + fine-tuned BERT), entity replacement, subject-conditioned inference, and daily sentiment aggregation, all as background tasks.

Fine-Tuning & Training

End-to-end fine-tuning pipeline: LoRA adaptation (~1M trainable params), LLM-based data labeling (3,500+ labeled triples), and an ablation study across 4 base models to select the best performer (macro F1 0.757).
