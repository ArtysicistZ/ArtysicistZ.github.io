AlphaOne

A full-stack financial sentiment analysis platform that turns Reddit noise into actionable ticker intelligence.
Reproducible data pipeline, NLP enrichment, and a real-time React dashboard.

Full-Stack NLP Pipeline Financial Sentiment
View on GitHub Back to Portfolio

The Vision

Retail investors are drowning in social media noise. AlphaOne cuts through it by building a reproducible data pipeline from Reddit to PostgreSQL, enriching every post with sentence-level NLP analysis, and surfacing ticker sentiment trends through an interactive dashboard.

The system ingests posts from financial subreddits, handles Reddit's aggressive rate limits with exponential backoff, and processes each post through a multi-stage NLP pipeline: spaCy tokenization, FinBERT sentiment inference, topic tagging, and daily aggregation for trend visualization.

Designed with production-grade data engineering principles — concurrent-safe, idempotent batch ingestion with content versioning ensures no duplicate or corrupted records, even under parallel worker loads.

Technical Highlights

  • 4-Service Architecture

    React frontend, Spring Boot API, Celery/Redis workers, and PostgreSQL — deployed as independent containers via Docker Compose for clean separation of concerns and easy scaling.

  • FinBERT Sentiment Analysis

    Sentence-level NLP pipeline using spaCy for tokenization and FinBERT for financial domain sentiment inference. Each sentence gets a positive/negative/neutral score with confidence.

  • Idempotent Data Pipeline

    Concurrent-safe batch ingestion with content versioning. Reddit posts are deduplicated, versioned, and enriched — ensuring the pipeline is fully reproducible from scratch.

  • Interactive Dashboard

    React frontend with real-time sentiment charts, ticker-level drill-downs, daily trend aggregation, and topic-based filtering for actionable insights.

Under The Hood

Backend API

Spring Boot serves the REST API with paginated endpoints for posts, sentiment scores, and aggregated trends. The API layer handles request validation, caching, and CORS configuration for the React frontend.

Worker Pipeline

Celery workers with Redis broker handle the heavy lifting: Reddit scraping with rate-limit backoff, NLP enrichment (spaCy + FinBERT), topic tagging, and daily sentiment aggregation — all running as background tasks.

Fine-Tune Pipeline

Currently designing a fine-tuning pipeline for a subject-oriented BERT model that will improve sentiment accuracy on financial-specific language and ticker mentions beyond the general FinBERT baseline.
